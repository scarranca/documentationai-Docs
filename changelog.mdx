## Upgrade Instructions

Follow these steps to update your Gigstack integration to the latest version.

<Steps>
  <Step title="Check Compatibility" icon="check-circle">

Review your current setup against the [migration guide](https://docs.gigstack.io/migration).

  </Step>
  <Step title="Update Webhooks" icon="settings">

````javascript
// Update your webhook endpoint to v1.2.0 format
app.post('/gigstack-webhook', async (req, res) => {
  const event = req.body;
  if (event.type === 'payment_intent.succeeded') {
    await gigstack.createCFDI({
      stripeEvent: event,
      apiKey: process.env.GIGSTACK_API_KEY
    });
  }
  res.json({received: true});
});
````

  </Step>
  <Step title="Test Automation" icon="play-circle">

Use the dashboard tester to simulate Stripe payments and verify CFDI issuance.

  </Step>
  <Step title="Monitor Logs" icon="activity">

Check the events log for any sync errors post-upgrade.

  </Step>
</Steps>

<Callout kind="tip">
For self-hosted setups, run `gigstack-cli upgrade` to apply the latest patches automatically.
</Callout>
