## Quick Guide

Use Markdown files to document your projects.

<Steps>
  <Step title="Go to your project">
    Select the project where you want to add documentation.
  </Step>
  <Step title="Click 'Add File'">
    Use the **Add File** option in the sidebar.
  </Step>
  <Step title="Choose 'Markdown'">
    Select **Markdown File** as file type.
  </Step>
  <Step title="Edit and save">
    Enter your content in the editor, then click **Save**.
  </Step>
</Steps>
