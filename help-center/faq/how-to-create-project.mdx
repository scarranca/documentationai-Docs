## Overview

To create a new project in Gigstack:

1. Click the **New Project** button on your dashboard.
2. Enter a project name and optional description.
3. Choose a workspace (if applicable).
4. Click **Create**.

## Tips

- Projects can be renamed later.
- Add documentation or invite team members once created.

<Callout kind="info">
New projects are private by default.
</Callout>
